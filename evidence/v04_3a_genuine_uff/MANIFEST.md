# 3a (UiO-66 + CO2) genuine-UFF evidence (v0.4.2 rebuild, 2026-06-21)

Promoted from validated production-count scratch runs (TASK 11/14, RASPA3) to committed evidence
(TASK 15), so the v0.4.2 strict-pass numbers trace to committed JSON. Parallel to the 6c
`evidence/v04_6c_olson_clean/` promotion.

## Why this rebuild
The originally-locked 3a recipe was generated by a RASPA3 input generator that hard-coded a
zeolite/cation LJ table and unconditionally stamped `source="UFF"` on the MOF framework atoms,
even though the light-atom values were not genuine UFF (framework C carried the Harris-Yung
CO2-carbon eps 28.1, O carried TraPPE-zeo 53.0, H a wrong eps; only the metals were genuine UFF).
This produced the false "UFF baseline under-binds" reading (as-locked dlogKH = -0.802). The
generator was fixed (`input_writer._uff_lj_K`, fail-loud; zeolite path byte-identical).

## Recipe (corrected)
- Framework: **genuine plain Rappe UFF (1992)** per element (C 52.84/3.431, O 30.19/3.118,
  H 22.14/2.571, Zr 34.72/2.783) + PACMOF2 training-set DDEC6 charges from the CIF.
- Guest: TraPPE/EPM2 CO2 as locked; Ewald; shifted-truncated; 12 A cutoff (as the published 2a/3a path).
- Structure: `RUBTAK01_SL_DDEC.cif` (all-silica UiO-66, PACMOF2 DDEC).

## Result (vs reference K_H 5.14, Q_st 26.5)
- Engine: **RASPA3 v3.0.29** (binary sha256 `2b4132becc0f38de...`), **12 seeds x 400,000 cycles**, T = 298.15 and 323.15 K.
- K_H(298) = 6.05 (median) / 6.06 (trimmed geomean) / 6.27 (geomean) mol/kg/bar; tight (low scatter).
- Q_st (two-point van't Hoff, 298/323) = 25.6 kJ/mol.
- dlogKH = +0.086, dQst = -0.89 -> **Tier-A STRICT PASS**.
- Full per-seed values + input sha256 in `results.json`; the run is reproducible from the committed inputs (force_field.json, simulation.json, CIF; sha256 recorded in results.json). For brevity the multi-MB full RASPA3 output logs are not vendored here; the per-seed Henry coefficients are in `results.json`.

## Caveat (RASPA3-only)
The native charged-Ewald estimator shows a large empirical discrepancy on these oblique MOF
primitive cells (~4x on HKUST-1; root cause not established -- see the manuscript limitation), so
it does **not** corroborate these numbers. The verdict rests on RASPA3 alone.

Superseded as-locked: dlogKH = -0.802 / dQst = -9.21 (retained in the case matrix as the documented
buggy case).
